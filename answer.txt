If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. 
Does JAVA allow you to do this?
--Có, Java cho phép tạo nhiều constructor với các tham số khác nhau (constructor overloading). 
Miễn là danh sách tham số của mỗi constructor là duy nhất, Java sẽ phân biệt được chúng.
